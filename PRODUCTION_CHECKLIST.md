# ✅ قائمة فحص الانتقال للإنتاج

## 🔧 التحضيرات الأساسية
- [ ] تحديث مفاتيح Moyasar الحقيقية في متغيرات البيئة
- [ ] تحديث عنوان الموقع الحقيقي في NEXT_PUBLIC_SITE_URL
- [ ] تفعيل Cloudflare Turnstile للحماية
- [ ] إضافة Resend API Key لإرسال الإيميلات
- [ ] رفع الكود النهائي للمستودع

## 🧪 اختبارات ما بعد النشر

### اختبار نظام الدفع:
- [ ] إنشاء طلب جديد كعميل
- [ ] تلقي عرض سعر من الادمن
- [ ] اختبار الموافقة على العرض (مع وبدون خدمات إضافية)
- [ ] إجراء عملية دفع حقيقية بمبلغ صغير (1 ريال مثلاً)
- [ ] التحقق من تحديث حالة الطلب بعد الدفع
- [ ] تلقي إيميل تأكيد الدفع

### اختبار النظام العام:
- [ ] تسجيل حساب جديد
- [ ] تسجيل دخول وخروج
- [ ] إضافة طلب جديد
- [ ] عرض لوحة التحكم
- [ ] وظائف الادمن
- [ ] إرسال الإيميلات

### اختبار الأمان:
- [ ] التحقق من عمل Cloudflare Turnstile
- [ ] اختبار الحماية من الدخول غير المصرح به
- [ ] فحص أمان نقل البيانات (HTTPS)

## 🔄 خطة العودة (Rollback)
في حالة مواجهة مشاكل:
1. العودة للكود السابق: `git revert HEAD`
2. إعادة نشر النسخة السابقة
3. التبديل لمفاتيح الاختبار مؤقتاً حتى حل المشكلة

## 📊 المراقبة بعد النشر
- مراقبة لوجز الأخطاء لأول 24-48 ساعة
- متابعة عمليات الدفع والتأكد من نجاحها
- مراقبة أداء الموقع وسرعة التحميل
- فحص تلقي الإيميلات بشكل صحيح
